# -File-Utility-App
  File Utility App is an all-in-one website that helps you convert files quickly and  easily. Whether you need to change a PDF to Word, PowerPoint to PDF, or  even convert images and videos, this app has you covered. You don’t need  multiple tools—everything you need is in one place.  Web Server: XAMPP (Apache + MySQL) 
 Languages: PHP, HTML, CSS, Python
